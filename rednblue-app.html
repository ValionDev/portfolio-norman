RednBlue App

Complete UX/UI development for RednBlue's mobile app, stewarding the design from wireframing through Zeplin handoff with close IT coordination.

Overview

RednBlue needed a mobile app that extended their web platform to mobile users. I stewarded the complete UX/UI development, from initial wireframes through high-fidelity design and developer handoff via Zeplin. The app shipped to the Apple App Store.

The Challenge

Translating a feature-rich web dashboard into a mobile experience that feels native and intuitive, not like a shrunken desktop app. Key challenges:

  • The web platform had complex data views that needed to be rethought for smaller screens
  • Users expected feature parity with the web while wanting a streamlined mobile experience
  • Tight coordination with the IT team was required for technical feasibility at every stage
  • The app needed to maintain RednBlue's brand identity while following platform conventions

Approach

  • Extensive wireframing: Started with low-fidelity wireframes to test information hierarchy and navigation patterns before investing in visual design
  • Mobile-first information architecture: Restructured the data views for progressive disclosure on mobile, surfacing key actions and summarizing dense data
  • Iterative prototyping: Built interactive prototypes to test navigation patterns and user flows with stakeholders before committing to final designs
  • Developer coordination: Worked closely with the IT team throughout, using Zeplin for spec handoff with organized component documentation and redlines

Key Design Decisions

Navigation Structure

Chose a tab-based navigation for primary sections with contextual actions surfaced within each view. This gave users quick access to core features without overwhelming the interface with options.

Dashboard Adaptation

The web dashboard's multi-column data views were reorganized into scrollable card-based layouts with expandable detail views. Key metrics got prominent placement, with deeper data one tap away.

Brand Consistency

Maintained RednBlue's visual identity while respecting iOS and Android platform conventions. Used the brand's color system and typography as anchors while adapting component patterns to feel native on each platform.

Outcome

The app shipped to the Apple App Store, giving RednBlue's clients mobile access to the platform. The design process, from wireframing through organized Zeplin handoff, set a workflow standard that the team continued using for future projects.
